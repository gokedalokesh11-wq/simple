🧩 HTML Elements

📘 What is an HTML Element?

An HTML element is the basic building block of a webpage. It tells the browser how to display content.

⚙️ Structure of an Element

Most HTML elements consist of:

  • Opening Tag → <tagname>
  • Content → Text or other elements
  • Closing Tag → </tagname>

Example:

<p>This is a paragraph</p>

🔍 Types of HTML Elements

  • Block Elements → Take full width (e.g., <div>, <p>, <h1>)
  • Inline Elements → Take only needed width (e.g., <span>, <a>, <img>)

⭐ Empty Elements

Some elements do not have closing tags. These are called empty elements.

  • <br> → Line break
  • <hr> → Horizontal line
  • <img> → Image

💡 Nested Elements

HTML elements can be placed inside other elements.

<p>This is <b>bold</b> text</p>

🚀 Why HTML Elements Matter?

  • They structure web pages
  • They define meaning of content
  • They help browsers render correctly
  • They improve SEO and accessibility
