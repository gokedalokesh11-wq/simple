What is HTML?

HTML stands for HyperText Markup Language.
It is used to create web pages and websites.

👉 HTML is not a programming language.
👉 It is a markup language used to structure content on the web.

Why HTML is Important?

HTML is the basic building block of every website.

With HTML you can:
✔ Create web pages
✔ Add text, images, videos, and links
✔ Build forms and tables
✔ Design website structure

How HTML Works

HTML uses tags to tell the browser what to display.

Example:
<h1> → Heading
<p> → Paragraph
