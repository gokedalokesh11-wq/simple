📘 What is HTML?

HTML stands for HyperText Markup Language.

It is used to create web pages and websites.

👉 HTML is not a programming language.
👉 It is a markup language used to structure content.

✨ Why HTML is Important?

✔ Create web pages
✔ Add text, images, videos
✔ Build forms and tables
✔ Structure websites

⚙️ How HTML Works

HTML uses tags to display content.

Example:
<h1> → Heading
<p> → Paragraph
